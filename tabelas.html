Para criar tabelas, usamos a tag table. É importante acrescentar o atributo border="numero" para que os limites da tabela apareçam.

Dentro dessa tag, o cabeçalho irá dentro da tag thead e o seu corpo dentro da tag tbody. Cada coluna do cabeçado vem dentro da tag th.

Para o corpo temos: Primeiro, definimos quantas linhas queremos ter com a tag tr. Em cada linha colocamos os dados que virão na coluna com a tag td. Exemplo (abra o código fonte):
